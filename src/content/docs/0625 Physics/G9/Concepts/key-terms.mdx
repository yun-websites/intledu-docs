## 关键术语

| 关键词     | 定义                                              | 单位 / 符号  |
| ----------- | ------------------------------------------------------- | -------------- |
| 质量 (m)    | 物体中物质的量。在任何地方都恒定。    | 千克 (kg)  |
| 重量 (W)  | 作用在质量上的重力。取决于位置。 | 牛顿 (N)     |
| 密度 (ρ) | 物质单位体积的质量。                    | kg/m³ or g/cm³ |
| 体积 (V)  | 物体或物质占据的空间。               | m³ or cm³      |
| 浮力    | 流体对浸没物体施加的向上的力。  | 牛顿 (N)     |
