## 矢量和标量

| 矢量                                              | 标量                                    |
| --------------------------------------------------- | ----------------------------------------- |
| 既有大小又有方向（指向某个方向） | 只有大小（大小 = 数字 + 单位） |
| 示例:                                               | 示例:                                     |
| - 位移                                      | - 时间                                    |
| - 重力场强度                      | - 体积                                  |
| - 速度                                          | - 距离                                |
| - 加速度                                      | - 能量                                  |
| - 重量                                            | - 质量                                    |
| - 力                                            | - 密度                                 |
|                                                     | - 温度                             |
|                                                     | - 速率                                   |

**位移**：从初始点到最终点的最短距离（或特定方向的距离）。  
**速度**：特定方向的速度。
