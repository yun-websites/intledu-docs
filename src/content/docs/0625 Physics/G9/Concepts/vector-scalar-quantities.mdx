## 矢量和标量

| Vector                                              | Scalar                                    |
| --------------------------------------------------- | ----------------------------------------- |
| Has both magnitude and direction (points somewhere) | Has magnitude only (size = number + unit) |
| 示例:                                               | 示例:                                     |
| - Displacement                                      | - Time                                    |
| - Gravitational field strength                      | - Volume                                  |
| - Velocity                                          | - Distance                                |
| - Acceleration                                      | - Energy                                  |
| - Weight                                            | - Mass                                    |
| - Forces                                            | - Density                                 |
|                                                     | - Temperature                             |
|                                                     | - Speed                                   |

**Displacement**：从初始点到最终点的最短距离（或特定方向的距离）。  
**Velocity**：特定方向的速度。
