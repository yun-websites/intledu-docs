## 矢量关键词

| Keyword             | 含义                                           |
| :------------------ | :-------------------------------------------- |
| Vector              | 具有大小和方向的量。                           |
| Scalar              | 只有大小的量。                                 |
| Magnitude           | 物理量的大小或数量。                           |
| Resultant           | 具有相同效果的组合单个矢量。                   |
| Component           | 矢量的垂直部分之一。                           |
| Perpendicular       | 成 90° 角。                                   |
| Scale drawing       | 使用比例尺表示大小的图表。                     |
| Pythagoras Theorem  | $R^2 = A^2 + B^2$。                            |
| Trigonometric Ratio | 边和角之间的关系。                             |
| Direction           | 矢量的方向。                                   |
| Bearing             | 从正北顺时针测量的角度。                       |
