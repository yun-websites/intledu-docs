## 力矩和力的转动效应

力的力矩是它围绕支点的转动效应。

**定义**  
力矩是力与到支点的垂直距离的乘积。

$$
\text{力矩} = F \times d
$$

其中 $ F $ 是力（单位为 N），$ d $ 是垂直距离（单位为 m），单位是 Nm。

**关键词**

| 关键词                | 含义                                         |
| ---------------------- | ------------------------------------------- |
| 力矩                 | 力的转动效应。                               |
| 支点                  | 发生转动的点。                               |
| 垂直距离 | 从支点到力的作用线的最短距离。               |
| 顺时针力矩       | 顺时针转动的力矩。                           |
| 逆时针力矩   | 逆时针转动的力矩。                           |
| 平衡            | 平衡 – 无转动。                              |
| 力                  | 作用在物体上的推或拉（单位为牛顿）。         |

**力矩原理**  
在平衡状态下，顺时针总力矩等于逆时针总力矩。

**平衡条件**

1. 物体上没有合力作用（$ F\_{\text{net}} = 0 $）。
2. 合力矩为零（顺时针总力矩 = 逆时针总力矩）。
