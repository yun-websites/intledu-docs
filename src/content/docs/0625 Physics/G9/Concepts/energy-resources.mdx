## 能源资源

Energy resources 提供能源来源。

### 可再生和不可再生

| 可再生                                                                                                   | 不可再生                                                                  |
| ----------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| <ul><li>Solar</li><li>Wind</li><li>Hydroelectric</li><li>Biofuel</li><li>Geothermal</li><li>Tidal</li></ul> | <ul><li>Nuclear</li><li>Fossil fuel</li><li>Coal</li><li>Natural gas</li></ul> |

### 基于太阳和非基于太阳

| 基于太阳                                                                                                                                                       | 非基于太阳                                                                  |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| <ul><li>Solar</li><li>Wind</li><li>Hydroelectric</li><li>Biofuel</li><li>Geothermal</li><li>Fossil fuel</li><li>Natural gas</li><li>coal</li><li>Wave</li></ul> | <ul><li>Nuclear</li><li>Fossil fuel</li><li>Geothermal</li><li>Tidal</li></ul> |

|                | 可再生资源                                                     | 不可再生资源                                                       |
| -------------- | ----------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| **定义** | 不会耗尽且通常无污染的能源 | 会耗尽且短时间内无法替代的能源 |
| **示例**   | Solar Wind Hydroelectric Geothermal Biofuel Tidal.                      | Nuclear Coal Natural Gas Fossil Fuel                                          |
