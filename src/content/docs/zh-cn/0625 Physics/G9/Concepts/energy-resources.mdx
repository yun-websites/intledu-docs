## 能源资源

能源资源提供能源来源。

### 可再生和不可再生

| 可再生                                                                                                   | 不可再生                                                                  |
| ----------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| <ul><li>太阳能</li><li>风能</li><li>水电</li><li>生物燃料</li><li>地热能</li><li>潮汐能</li></ul> | <ul><li>核能</li><li>化石燃料</li><li>煤炭</li><li>天然气</li></ul> |

### 基于太阳和非基于太阳

| 基于太阳                                                                                                                                                       | 非基于太阳                                                                  |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| <ul><li>太阳能</li><li>风能</li><li>水电</li><li>生物燃料</li><li>地热能</li><li>化石燃料</li><li>天然气</li><li>煤炭</li><li>波浪能</li></ul> | <ul><li>核能</li><li>化石燃料</li><li>地热能</li><li>潮汐能</li></ul> |

|                | 可再生资源                                                     | 不可再生资源                                                       |
| -------------- | ----------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| **定义** | 不会耗尽且通常无污染的能源 | 会耗尽且短时间内无法替代的能源 |
| **示例**   | 太阳能 风能 水电 地热能 生物燃料 潮汐能.                      | 核能 煤炭 天然气 化石燃料                                          |
