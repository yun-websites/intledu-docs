## 有效数字（规则）

1. 测量数字中的所有非零数字都是有效的。  
   (1–9 都是有效的)  
   示例：235.52 → **5 s.f.**

2. 用于占位的前导零（小数点前）**不**是有效的。  
   (在 1–9 之前，0 不是有效的)  
   示例：0.0073 → **2 s.f.**

3. 非零数字之间的零是有效的。  
   (在 1–9 之间，0 是有效的)  
   示例：2300.1 → **5 s.f.**

4. 非零数字后的零（且在小数点后）是有效的。  
   示例：0.0010 → **2 s.f.**

5. 小数点后的零是有效的。  
   示例：0.00 → **2 s.f.**

6. 有效数字之间的任何数字都是有效的。  
   示例：205004 → **6 s.f.**
