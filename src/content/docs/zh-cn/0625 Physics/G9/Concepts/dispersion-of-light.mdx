## 光的色散 – 总结

| Key Term               | 含义                                                                                   |
| ---------------------- | ------------------------------------------------------------------------------------------- |
| Dispersion             | 白光通过棱镜时分解成不同颜色的现象。                                                     |
| White Light            | 由许多波长（多种颜色）组成的混合物。                                                     |
| Monochromatic Light    | 由单一波长（一种颜色）组成的光，例如激光。                                               |
| Why Dispersion Happens | 不同波长在玻璃中以不同速度传播，并以不同的量折射。                                         |
| Colour Bending         | 红色弯曲最少（波长最长）。紫色弯曲最多（波长最短）。                                      |

| 可见光谱顺序                                 | 记忆辅助 |
| ------------------------------------------------------ | ---------- |
| Red → Orange → Yellow → Green → Blue → Indigo → Violet | ROYGBIV    |

**考试提示：** 解释色散时，提及波长、在玻璃中的不同速度以及不同的折射角度。

- 光进入较密介质时向法线弯曲。
- 光进入较疏介质时远离法线弯曲。
- 频率保持不变。
- 波长发生变化。
