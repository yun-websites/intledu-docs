## 金属

**核心思想：容易失去电子 → 导电性 + 延展性**

- 大多数为固体（汞除外）
- 高熔点/沸点
- 坚硬、密度大
- 良好的导体（电和热）
- 可锻造（可锤打成薄片）
- 可延展（可拉成 wire）
- 通常为灰色（例外：银、金）
- 有金属光泽

## 非金属

**核心思想：倾向获得电子 → 绝缘性 + 脆性**

- 固体或气体（溴为液体除外）
- 低熔点/沸点
- 低密度
- 不良导体
- 脆性（如果是固体）
- 无金属光泽

## 过渡金属

**核心思想：复杂的电子结构 → 多变价 + 催化作用**

- 高熔点/沸点
- 坚硬且密度大
- 颜色多变
- 可变氧化态
- 常作为催化剂
