## 常见离子的电荷

| 电荷 | 离子                                               |
| :--: | :------------------------------------------------- |
|  1+  | $Ag^+$ 和所有第 I 族离子                           |
|  2+  | $Zn^{2+}$, $Cu^{2+}$, $Fe^{2+}$ 和所有第 II 族离子 |
|  3+  | $Fe^{3+}$ 和所有第 III 族离子                      |
|  2-  | $SO_4^{2-}$, $CO_3^{2-}$ 和所有第 VI 族离子        |
|  1-  | $NO_3^-$, $OH^-$, $HCO_3^-$ 和所有第 VII 族离子    |

## 确定化学式

化合物的总电荷 = 0。  
总正电荷 = 总负电荷。  
如果需要多个复杂离子，用括号括起来 ()。
