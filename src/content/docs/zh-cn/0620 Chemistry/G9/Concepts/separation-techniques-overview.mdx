本节介绍实验室中用于分离混合物的主要方法。技术的选择取决于组分的物理性质（溶解度、沸点、磁性等）。

## 主要分离方法

- **过滤**  
  分离不溶性固体与液体。

- **结晶**  
  通过蒸发和冷却从溶液中获得可溶性固体。

- **磁分离**  
  分离磁性固体与非磁性固体。

- **简单蒸馏**  
  分离沸点差异较大（>70°C）的互溶液体。

- **分馏**  
  分离沸点相近（\<70°C 差值）的互溶液体。

- **纸色谱法**  
  基于不同溶解度和吸附性分离和鉴定组分（尤其是染料或有色物质）。

## 何时使用每种方法

| 混合物类型               | 推荐方法 | 利用的关键性质 |
| ------------------------ | -------- | -------------- |
| 不溶性固体 + 液体        | 过滤     | 溶解度         |
| 溶解在液体中的可溶性固体 | 结晶     | 溶解度和温度   |
| 磁性 + 非磁性固体        | 磁分离   | 磁性           |
| 两种液体（沸点差异大）   | 简单蒸馏 | 沸点           |
| 两种液体（沸点相近）     | 分馏     | 沸点 + 表面积  |
| 染料 / 有色化合物混合物  | 纸色谱法 | 溶解度和吸附性 |
