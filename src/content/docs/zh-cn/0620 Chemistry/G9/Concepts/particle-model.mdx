**核心：物质性质 = 粒子排列 + 粒子运动 + 作用力**

这是整章的"总开关"，后面所有现象都能从这里推出来。

## 物质状态

### 固体

**机制：强作用力 + 固定位置**

- 紧密排列
- 规则的晶格排列
- 在固定位置振动
- 高密度
- 不可压缩
- 不流动

👉 本质一句话：
**粒子被"锁死"，只能抖，不能跑**

### 液体

**机制：作用力减弱 + 可滑动**

- 紧密排列（但不规则）
- 可以相互移动
- 固定体积
- 无固定形状
- 可流动

👉 本质一句话：
**粒子开始"挣脱"，可以互相绕开**

### 气体

**机制：几乎无作用力 + 自由运动**

- 距离较远
- 随机运动
- 无固定形状或体积
- 易压缩
- 低密度

👉 本质一句话：
**粒子完全自由，像一群喝了咖啡的电子**

### 气体行为

- 温度 ↑ → 粒子运动更快 → 体积 ↑
- 压力 ↓ → 粒子扩散 → 体积 ↑

👉 核心不是公式，而是：
**空间 vs 动能 的博弈**
