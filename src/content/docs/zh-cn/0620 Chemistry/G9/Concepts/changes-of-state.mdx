## 能量视角

这是关键统一逻辑：

- **吸热** → 粒子获得能量 → 更自由
- **放热** → 粒子失去能量 → 更受限

## 相变

### 吸热

- 熔化：固体 → 液体
- 沸腾：液体 → 气体
- 蒸发：液体 → 气体（仅表面）
- 升华：固体 → 气体

### 放热

- 凝结：气体 → 液体
- 凝固：液体 → 固体

## 沸腾与蒸发

- 沸腾 → 整体发生（bulk）
- 蒸发 → 只在表面

👉 深层区别：
**是否所有粒子都有足够能量逃逸**
