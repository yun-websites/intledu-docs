An element name is taken from the Periodic Table.

## Monatomic elements

Symbol is the same as in the Periodic Table (all metals and noble gases).

## Diatomic elements

Formula is symbol + subscript 2 ($X_2$), e.g., $H_2$, $N_2$, $O_2$ and all halogens.

## Polyatomic elements

Formula is symbol + subscript marker, e.g., $P_4$, $S_8$.
