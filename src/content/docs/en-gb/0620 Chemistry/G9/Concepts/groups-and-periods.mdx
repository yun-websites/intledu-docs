**Group Number** → Outer electron number  
**Period Number** → Shell number

Groups: elements with similar chemical properties (same outer electrons)  
Periods: elements with the same number of occupied shells

## Noble Gases (Group VIII)

- Full outermost shell
- Very stable electron arrangement
- Unreactive (noble) gases
