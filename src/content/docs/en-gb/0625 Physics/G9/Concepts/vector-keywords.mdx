## Vector Keywords

| Keyword             | Meaning                                       |
| :------------------ | :-------------------------------------------- |
| Vector              | Quantity with magnitude and direction.        |
| Scalar              | Quantity with magnitude only.                 |
| Magnitude           | Size or amount of a physical quantity.        |
| Resultant           | Combined single vector having same effect.    |
| Component           | One of the perpendicular parts of a vector.   |
| Perpendicular       | At 90° angle.                                 |
| Scale drawing       | Diagram using a scale to represent magnitude. |
| Pythagoras Theorem  | $R^2 = A^2 + B^2$.                            |
| Trigonometric Ratio | Relation between sides and angles.            |
| Direction           | Orientation of a vector.                      |
| Bearing             | Angle measured clockwise from north.          |
