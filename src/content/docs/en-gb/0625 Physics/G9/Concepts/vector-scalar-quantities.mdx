## Vector and Scalar Quantities

| Vector                                              | Scalar                                    |
| --------------------------------------------------- | ----------------------------------------- |
| Has both magnitude and direction (points somewhere) | Has magnitude only (size = number + unit) |
| Examples:                                           | Examples:                                 |
| - Displacement                                      | - Time                                    |
| - Gravitational field strength                      | - Volume                                  |
| - Velocity                                          | - Distance                                |
| - Acceleration                                      | - Energy                                  |
| - Weight                                            | - Mass                                    |
| - Forces                                            | - Density                                 |
|                                                     | - Temperature                             |
|                                                     | - Speed                                   |

**Displacement**: Shortest distance from initial point to final point (or distance in a specific direction).  
**Velocity**: Speed in a specific direction.
