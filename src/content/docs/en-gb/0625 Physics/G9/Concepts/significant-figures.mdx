## Significant Figures (Rules)

1. All non-zero digits in a measured number are significant.  
   (1–9 all are significant)  
   Example: 235.52 → **5 s.f.**

2. Leading zeros that are holding the place of a decimal are **NOT** significant.  
   (Before 1–9, 0 is not significant)  
   Example: 0.0073 → **2 s.f.**

3. Zeros between non-zero numbers are significant.  
   (Between 1–9, 0 is significant)  
   Example: 2300.1 → **5 s.f.**

4. Zero after a non-zero number (and after decimal) are significant.  
   Example: 0.0010 → **2 s.f.**

5. Zeros after a decimal point are significant.  
   Example: 0.00 → **2 s.f.**

6. Any digits between significant figures are significant.  
   Example: 205004 → **6 s.f.**
