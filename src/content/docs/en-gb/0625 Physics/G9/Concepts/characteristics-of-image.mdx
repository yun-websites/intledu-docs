## Image Produced by Converging Lenses

| Object Position | Image Type | Image Direction          | Image Size                   |
| --------------- | ---------- | ------------------------ | ---------------------------- |
| at > 2F         | Real       | Inverted (Upside down)   | Diminished (Smaller)         |
| at 2F           | Real       | Inverted (Upside down)   | Same Size as Object          |
| at < 2F         | Real       | Inverted (Upside down)   | Enlarged (Larger)            |
| at F            | Real\*     | Inverted\* (Upside down) | Enlarged (Image of Infinity) |
| at < F          | Virtual    | Upright                  | Enlarged (Larger)            |

**Note**

When an object is placed at the focal point (F) of a converging lens, the emerging rays become parallel.

Since parallel rays do not converge, no image is formed at any finite distance. The image is therefore said to be at infinity.
